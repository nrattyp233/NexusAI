# NexusAI | Autonomous Growth Engine

> **Ver 11.0.1** - Operational Status: GREEN

## ◼ System Overview
NexusAI is a high-performance, autonomous content generation and market trend analysis dashboard designed for the Micro-SaaS sector. Built on a modern React architecture and powered by **Google's Gemini 2.5 Flash** neural models, it provides users with "Business-in-a-box" capabilities for copywriting and strategic intelligence.

## ◼ Core Capabilities

### 1. Command Center (Dashboard)
- **Real-time Visualization:** Integrated `Recharts` for engagement and growth trajectory tracking.
- **Neural Insights:** Automated executive summaries of market data powered by AI.
- **Operational Metrics:** Live tracking of generated content value and efficiency rates.

### 2. Neural Forge (Generator)
- **Multi-Vector Generation:** Specialized prompts for LinkedIn, Twitter, Blog, and Email strategy.
- **History Architecture:** LocalStorage persistence layer to save, archive, and retrieve past generations.
- **Instant Deployment:** Copy-to-clipboard functionality and rapid regeneration workflows.

### 3. System Config
- **Identity Protocol:** User role and profile management.
- **Data Sovereignty:** Full JSON export functionality for data backup and migration.
- **Persistence:** Local client-side database simulation (No SQL setup required for demo).

## ◼ Technical Architecture

- **Frontend:** React 19 + TypeScript (Strict Mode)
- **Styling:** Tailwind CSS (Cyberpunk/Dark Mode aesthetic)
- **AI Core:** `@google/genai` SDK (Gemini 2.5 Flash)
- **Icons:** Lucide-React
- **State Management:** React Hooks + LocalStorage API

## ◼ Deployment Protocol

### Prerequisites
- Node.js v18+
- Google Gemini API Key

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure Environment:
   - Create a `.env` file in the root.
   - Add your API key: `REACT_APP_API_KEY=your_gemini_key_here` (Note: The current build handles keys via process.env or direct injection for demo purposes).

### Start Development Server
```bash
npm start
```
